# CS-594-EasyCloud:A Digital Cloud App
Easy Cloud is an android application for the users of Digital Ocean.
Using this application, one can create and manage the droplets from any remote locations.
Easy Cloud implements the basic functionalities such as:
1) Registration using EmailId,Token and Password
2) Login on providing correct authentication
3) List of all the droplets associated with the particular token on Digital Ocean Website
4) Create new droplet with options of Droplet Name,Size,Type of Operating System,Datacenter Location
5) Detailed view of particular droplet
6) More additional features such as Rename Droplets,Resize them also along with Enabling and Disabling backups.
